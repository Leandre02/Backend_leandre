## Projet Final : Backend - Développement Web 3 - Automne 2025

# API Bibliothèque Personnelle 

API REST pour gérer une bibliothèque personnelle de livres avec authentification JWT.

# Instructions
1. Cloner le dépôt backend sur votre machine.  
2. Importez le fichier bibliotheque.json dans votre BD MongoDB.
3. Installez les modules node avec `npm install`.
4. Créer un fichier `.env` à la racine du projet et configurer les variables d'environnements comme indiqué dans .env.example
5. Exécutez le serveur avec `npm run dev`.
6. Créer un utilisateur de test via l’API
7. Utiliser **Bruno** pour tester votre API (voir dossier Bruno_leandre)
